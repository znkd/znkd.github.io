## 欢迎

欢迎来到我的博客！这是我使用 Hugo 和 PaperMod 主题创建的第一篇文章。

## 功能

- 响应式设计，适配手机和桌面设备
- 亮色/暗色主题切换
- 代码高亮
- 文章分类和标签
- 阅读时间和字数统计

## 开始写作

要创建新文章，可以运行：

```bash
hugo new posts/文章标题.md
```

## 本地预览

启动本地服务器：

```bash
hugo server -D
```

然后在浏览器中访问 http://localhost:1313

## 构建静态网站

```bash
hugo -D
```

构建后的文件会保存在 `public` 目录下。

希望您喜欢这个博客主题！
